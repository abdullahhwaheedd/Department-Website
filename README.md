Computer Science Department Website

A simple static multi-page website built using HTML and internal CSS as a beginner-level group project.

About the Project:

This project is a basic department website designed to practice fundamental web development concepts.
All pages are connected through navigation, and styling is written inside each HTML file using internal CSS.

The main focus of this project is to understand:

Page structure using HTML

Navigation between multiple pages

Styling using CSS

Organizing files inside a single project folder

Technologies Used

HTML5

CSS3 (Internal CSS)

 Features:

Clean and simple layout

Multiple linked pages

Internal CSS styling

Beginner-friendly and easy to understand structure

 Purpose:

This project was created for learning and academic practice.
It is intended to strengthen core concepts of HTML and CSS without using any frameworks or JavaScript.
